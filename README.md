# Orbital Mechanics: Lagrange Points Explorer

A 3D interactive simulation of Lagrange Points in orbital mechanics, built with React, Three.js (Fiber), and Tailwind CSS.

![Lagrange Points Simulation](https://img.shields.io/badge/Status-Active-success)
![License](https://img.shields.io/badge/License-MIT-blue)

## Features

*   **Interactive 3D Scene**: Visualize the 5 Lagrange points in a two-body system.
*   **Adjustable Mass Ratio (μ)**: dynamically change the mass ratio between the two primary bodies to see how L-points shift (from Earth-Sun like systems to Binary Star systems).
*   **Potential Field Visualization**: Toggle the visualization of the gravitational potential field.
*   **Responsive UI**: Modern, cinematic UI built with Tailwind CSS and Framer Motion.

## Technologies Used

*   **React 19**
*   **Vite**
*   **React Three Fiber (@react-three/fiber)** & **Drei**
*   **Tailwind CSS 4**
*   **Framer Motion**

## Getting Started

### Prerequisites

*   Node.js (v18+ recommended)
*   npm

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/your-username/lagrange-points-simulation.git
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Run the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is configured for deployment to GitHub Pages.

```bash
npm run deploy
```
